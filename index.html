HTML

  • HTML (HyperText Markup Language) is the code that is used to structure a web page and its content
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The Img element contains information about the images that are displayed on the webpage
  • The hyperlink element creates links to the same webpage or other webpages
  • The footer element appears at the bottom of the page and normally contains the author information, copyright information, contact information, the sitemap and navigation options

CSS

  • CSS (Cascading Style Sheets) is the code that styles web content
  • CSS is a style sheet language. CSS is what you use to selectively style HTML elements
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Github is a web-based interface that uses Git
  • Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency
  • Git is the open source version control software that lets multiple people make separate changes to web pages at the same time
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a versatile, beginner friendly and powerful programming language that can add interactivity to a website
  • JavaScript controls the behavior of responses when buttons are pressed or with data entry on forms; with dynamic styling; with animation, etc
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
